# Chatting Service (Console Version)

<br>

## :pencil2: 요약

* Node.js의 net 모듈(TCP)을 사용해서 채팅 서비스를 구현했다.

* Socket을 이해하기 위해서 시작했지만, 더 공부해야할 필요가 있다.

* 로그인 기능을 추가 했고, crypto 모듈을 사용해서 비밀번호를 암호화 했다.

* .csv 파일 시스템으로 Database를 구현했다.
